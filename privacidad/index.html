1. Introducción y Compromiso

Goatify APP ("Nosotros", "La Plataforma") se compromete a proteger su privacidad con un enfoque de "Security First". Esta política detalla exhaustivamente la recopilación, uso y protección de la información personal de los usuarios de nuestros servicios de productividad y automatización IA. Al utilizar Goatify, usted acepta las prácticas descritas en este documento legal.

2. Datos que Recopilamos

Para el funcionamiento óptimo de los Agentes IA y el Gestor de Proyectos, recopilamos estrictamente lo necesario:

Información de Cuenta

Nombre, Email, Foto de perfil (Procesado vía Google Auth seguro).

Datos Operativos

Proyectos, tareas, notas y documentos que usted crea activamente.

Integraciones

Calendarios, eventos y disponibilidad (únicamente bajo su consentimiento explícito).

Datos Técnicos

Dirección IP y logs de seguridad (retenidos máx. 30 días).

3. Uso de Permisos de Google (Scopes)

Goatify solicita permisos específicos (scopes) para automatizar su agenda. A continuación, desglosamos cada permiso solicitado:

Permiso (Scope) Propósito Específico del Uso
.../auth/userinfo.email Autenticación única (SSO), creación de cuenta segura y comunicaciones críticas.
.../auth/calendar.events Permite a los Agentes IA agendar reuniones, detectar conflictos de horario y sincronizar hitos de proyecto.
.../auth/calendar.readonly Permite al asistente "Shivo" leer disponibilidad para responder consultas ("¿Qué tengo hoy?") sin riesgo de edición.

Cumplimiento con la Política de Google API

El uso de Goatify APP y la transferencia a cualquier otra aplicación de la información recibida de las API de Google se adherirá estrictamente a la Política de Datos de Usuario de los Servicios de API de Google, incluidos los requisitos de Uso Limitado.

"Goatify's use and transfer to any other app of information received from Google APIs will adhere to Google API Services User Data Policy, including the Limited Use requirements."

4. IA Ética y Procesamiento de Datos

Nuestros modelos de Inteligencia Artificial procesan sus datos exclusivamente para brindarle el servicio solicitado (ej. resumir una nota, agendar una cita).

Aislamiento de Datos

Sus datos personales y de negocio NO se utilizan para entrenar los modelos públicos de nuestros proveedores de IA. Utilizamos instancias privadas donde sus datos están aislados y son efímeros para el proceso de inferencia.

5. Infraestructura y Seguridad Física

Nuestra infraestructura está diseñada para ser resiliente y segura desde el diseño:

Proveedores Tier-1

Alojados en Google Cloud Platform (GCP) y Amazon Web Services (AWS).

Certificaciones

Centros de datos con ISO 27001, SOC 2 Tipo II y PCI-DSS Nivel 1.

Redundancia

Backups automáticos en múltiples zonas geográficas.

6. Cumplimiento Internacional

Goatify cumple con las regulaciones internacionales más estrictas (GDPR & CCPA):

Roles de Datos

Actuamos como Controlador para registro y facturación, y Procesador para su contenido.

Transferencia

Uso de Cláusulas Contractuales Estándar (SCC) para transferencias internacionales.

7. Respuesta ante Incidentes

Mantenemos un equipo de seguridad activo 24/7. En caso de una brecha:

  • Notificación a autoridades en máx. 72 horas.
  • Aviso inmediato a usuarios afectados.
